Do · Don't

Do

  • Один accent (red) — только active CTA
  • Selected = sky-50 + sky-500 left bar
  • Числа в правой колонке, tnum
  • Status — chip lowercase из закрытого набора
  • Hover — bg-hover, не border-color
  • Empty — одно предложение, без эмодзи

Don't

  • Selected красным (зарезервирован для CTA)
  • Status заглавными как Badge
  • Кнопки действий в каждой строке (только ··· меню)
  • Иллюстрации/эмодзи в empty state
  • Padding ячейки 24+ — это data UI, не landing
  • Вложенные градиенты или цветные тени
