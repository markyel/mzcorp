# 08 · Print (печать КП, счёта, акта)

> Документы на бумагу или PDF: коммерческие предложения, счета, договоры, акты, спецификации, упаковочные листы.

## Когда применять

- «печать КП», «счёт на оплату», «акт сверки»
- «PDF на бланке», «коммерческое предложение»
- «спецификация поставки», «упаковочный лист»

Это **не email-шаблоны**. Email — для inbox клиента (см. [06-email-templates.md](06-email-templates.md)). Print — для подписи / печати / приложения к договору. Часто email **прикладывает** print-PDF как вложение.

## Принципы

1. **A4 формат** — 210×297mm, поля 20mm/15mm/15mm/15mm (Т/L/R/B). Альбомный — только для спецификаций > 10 колонок.
2. **Чёрный текст на белом**. Accent красный — только реквизиты «Оплатить до» в счёте и подпись-печать. Никакого sky/emerald/amber.
3. **Шрифт** — sans 11pt body, mono 11pt для номеров/артикулов. Times и serif — нет (CRM не делает «солидно», CRM делает «чисто и быстро»).
4. **Линии** — 0.5pt чёрные. Никаких тонких серых рамок (не пропечатываются).
5. **Numbers** — табулярные всегда. `1 234,56 ₽` (русский: пробел-разделитель, запятая-десятая).
6. **Один документ — один логический объект.** Спецификация на 200 позиций — продолжается на стр. 2, 3, 4. Не сжимай шрифт.

## Header (бланк)

Левая колонка — логотип 30×30mm.
Правая колонка — юр. блок 90mm:

```
ООО «МАШЗАПЧАСТЬ» (бренд MyLift)
123022, Москва, ул. ……………, д. 12, оф. 5
ИНН 7733123456 · КПП 773301001 · ОГРН 1027700000000
тел. +7 495 123-45-67 · zakaz@mylift.ru · mylift.ru
р/с 40702810…  в ПАО «……»  БИК 044525000  к/с 30101810…
```

- Шрифт 9.5pt, line-height 1.35.
- Header повторяется на каждой странице.

## Title block

Между header и body:

```
                       КОММЕРЧЕСКОЕ ПРЕДЛОЖЕНИЕ № MZ-21384
                              от 22 ноября 2025 г.

   Заказчик: ТД «Лифт-Сервис» (ООО)
   ИНН/КПП:  7701234567 / 770101001
   Основание: запрос от 19.11.2025, бренд KONE
   Срок:     до 28.11.2025
```

- Тип документа — uppercase 14pt semibold по центру.
- Реквизиты слева, 11pt regular, label `--fg-2` (или просто bold). Колонка лейблов фикс ~28mm.

## Документы (закрытый набор)

### A · Коммерческое предложение

**Body — таблица позиций**:

| № | Артикул | Наименование | Бренд | Кол-во | Ед | Цена без НДС | Сумма без НДС |
|---|---|---|---|---|---|---|---|
| 1 | KM5071732G01 | Канатоведущий шкив D320 | KONE | 4 | шт | 34 200,00 | 136 800,00 |
| 2 | … | … | … | … | … | … | … |

- Заголовок таблицы 9pt uppercase fg-2 medium, фон `#F4F6F8`.
- Тело 10.5pt regular, num колонки tnum right-aligned.
- Hairline 0.4pt между строками.
- На разрыве страницы — повтор `<thead>`. CSS: `thead { display: table-header-group; }`.

**Totals** (справа, 70mm):

```
                              ИТОГО без НДС:    142 800,00 ₽
                              НДС 20%:           28 560,00 ₽
                              ИТОГО с НДС:      171 360,00 ₽
```

- Сумма прописью под totals: `Сто семьдесят одна тысяча триста шестьдесят рублей 00 копеек`. 9.5pt italic fg-2.

**После totals**:

```
УСЛОВИЯ
- Срок поставки: 3 рабочих дня с момента оплаты
- Условия оплаты: 100% предоплата
- Самовывоз с склада в Москве; доставка — по согласованию
- КП действительно до 28.11.2025
```

**Подписи** (нижняя треть):

```
От поставщика                                    От заказчика

_____________________ /Иванов И.И./        _____________________ / ________/
       М.П.                                          М.П.
```

### B · Счёт на оплату

Совмещённый формат (бланк-счёт). Особенности:

- Title: `СЧЁТ № MZ-21384/01 от 22.11.2025` крупно.
- В реквизитах банка — рамка 0.4pt с подписью `Образец заполнения платёжного поручения`.
- Срок оплаты: красным `--accent` 12pt semibold: `Оплатить до 28.11.2025`.
- Подпись бухгалтера справа от подписи руководителя:
  ```
  Руководитель _______________ /Иванов И.И./   Бухгалтер _______________ /Петрова О.О./
  ```

### C · Акт сверки взаиморасчётов

- Title: `АКТ СВЕРКИ ВЗАИМОРАСЧЁТОВ за период с … по …`.
- Двухколоночная таблица: левая колонка от поставщика, правая — от заказчика. Каждая колонка имеет `Дебет / Кредит / Сальдо`.
- В подвале: «Сальдо в пользу …» прописью + цифрой.
- Подписи с расшифровкой ФИО и должности.

### D · Спецификация / упаковочный лист

- Альбомная ориентация если > 8 колонок.
- Колонки: №, артикул, описание, бренд, кол-во, упак, вес кг, объём м³, место №, примечание.
- Footer: `Сдал ___________  Принял ___________  Дата ___________`.

## Layout правила (CSS for print)

```css
@page { size: A4; margin: 20mm 15mm 15mm 15mm; }
body  { font: 11pt/1.4 'Inter', system-ui, sans-serif; color:#000; background:#fff; }
.mono { font-family: 'JetBrains Mono', ui-monospace, monospace; }

/* page-break controls */
.no-break    { break-inside: avoid; page-break-inside: avoid; }
.break-before{ break-before: page; }

/* table sticky header on each printed page */
table.docs thead { display: table-header-group; }
table.docs tfoot { display: table-footer-group; }
table.docs tr    { break-inside: avoid; }

/* footer page numbers — running header/footer  */
@page { @bottom-right { content: "стр. " counter(page) " / " counter(pages); font-size: 9pt; color:#5C6975; } }

/* prevent backgrounds being silently dropped */
* { -webkit-print-color-adjust: exact; print-color-adjust: exact; }
```

## Numbers · форматирование

| Что | Формат |
|---|---|
| Деньги | `142 800,00 ₽` (пробел-тысячи, запятая-копейки, ₽ через NBSP) |
| Сумма прописью | `Сто сорок две тысячи восемьсот рублей 00 копеек` |
| ИНН | `7701234567` mono |
| КПП | `770101001` mono |
| Дата | `22 ноября 2025 г.` body, `22.11.2025` mono в таблицах |
| Артикул | mono, верхний регистр как в каталоге |
| Кол-во | tnum, без trailing zeros (`4`, не `4,00` если шт; `4,500` если кг) |

## Copywriting

- **Title** — uppercase, без знака `№` на новой строке. `КОММЕРЧЕСКОЕ ПРЕДЛОЖЕНИЕ № MZ-21384`.
- **Заказчик** — полное юр.название с организационной формой: `ТД «Лифт-Сервис» (ООО)`.
- **Срок** — конкретная дата, не `в течение недели`.
- **Подпись блок** — должность · подчёркивание-линия 60mm · `/ФИО/`. Не `(подпись)`, не `[подпись]`.
- **Печать** — `М.П.` маленькими прописными под линией подписи, по центру.
- **Sigle line conditions** — глагол + объект, точка в конце:
  - `Срок поставки: 3 рабочих дня с момента оплаты.`
  - `Условия оплаты: 100% предоплата.`

## Do / Don't

| ✅ Do | ❌ Don't |
|---|---|
| Чёрный на белом, 0.5pt линии | Цветной фон/границы |
| Header с реквизитами повторяется | Header только на 1-й странице |
| `thead` `display: table-header-group` | Таблица без повтора заголовка |
| Сумма прописью italic 9.5pt | Сумма прописью КАПСОМ |
| Дата `22 ноября 2025 г.` | Дата `22/11/25` или `Nov 22, 2025` |
| Срок оплаты в счёте — accent red | Все суммы accent red |
| `М.П.` под линией подписи | Картинка-печать |

## Edge cases

- **Длинное наименование позиции** — wrap внутри ячейки; высота ячейки растёт; страница не ломается, table-row `break-inside: avoid`.
- **Таблица не помещается** — переносится на следующую страницу с повтором thead. Заголовок документа НЕ повторяется, только реквизитный header (если включён в `running header`).
- **Нет НДС** (УСН) — строка `НДС: не облагается на основании п.2 ст.346.11 НК РФ`. Сумма прописью считается от итого без НДС.
- **Скидка** — отдельной строкой в totals: `Скидка 5%: −7 140,00 ₽`. Знак `−` (true minus, не дефис).
- **Альтернативные позиции** в КП — две таблицы, между ними подзаголовок 12pt semibold: `Вариант 1 · оригинал` / `Вариант 2 · аналог`.
- **Подпись pdf-EDS** — placeholder подпись блок остаётся, рядом текст `Документ подписан электронной подписью. ID: …`.
- **Печать из браузера** — пользователь должен видеть кнопку `Печать / PDF` в шапке экрана, она вызывает `window.print()` после применения print-CSS. Скрин внутри page без обрамления экрана CRM (`@media print { .topbar, .rail, .actions { display:none; } }`).
