# Глоссарий

Короткий словарь терминов, которые встречаются в карточках, чатах команды и в логах системы.

## Бизнес-термины

| Термин | Что значит |
|---|---|
| **Заявка** | Запрос клиента на покупку запчастей. Создаётся автоматически из письма на общий ящик. |
| **КП** | Коммерческое предложение. PDF, который вы отправляете клиенту с ценами и позициями. |
| **Счёт** | Документ для оплаты, выставляется после согласования КП. |
| **Refresh цен** | Запрос актуальной цены у поставщика на конкретную позицию. |
| **Sticky** | Правило «один клиент или артикул — один и тот же менеджер». Сохраняет историю отношений. |
| **Round-robin** | Распределение новых заявок по очереди между свободными менеджерами с учётом нагрузки. |
| **SLA** | Лимит времени на следующий шаг по заявке. По умолчанию 24 рабочих часа. |
| **Attention** | Маркер, что от менеджера ждут действия. Поднимается системой автоматически. |
| **Pool / пул** | Все активные заявки менеджера. |

## Роли

| Роль | Что делает |
|---|---|
| **Менеджер** | Ведёт свой пул заявок, общается с клиентами, готовит КП. |
| **РОП** | Управляет распределением, переподчиняет, видит дашборды, настраивает правила почты. Также ведёт свои заявки наравне с менеджером. |
| **Секретарь** | Контролирует, что почта попала в нужные руки. Без операционной работы с заявками. |
| **Директорат** | Аналитика, кураторство базы знаний, стратегические решения. |

## Технические сокращения

| Сокращение | Расшифровка |
|---|---|
| **CRM** | Customer Relationship Management. Эта система. |
| **IMAP** | Протокол, по которому система читает почту с Yandex 360. |
| **SMTP** | Протокол, по которому система отправляет письма. |
| **KB** | Knowledge Base. База знаний с типизированными артикулами и брендами. |
| **AI / LLM** | Используется для классификации писем, парсинга позиций и категоризации каталога. |
| **pgvector** | Подсистема PostgreSQL для семантического поиска по векторным представлениям. |

## Статусы заявок

Полный список — в гайде «[Работа с заявкой](/docs/manager/request-lifecycle)». Здесь — краткая шпаргалка:

```
Новая → В работе → Жду клиента → КП отправлено →
Согласован / ждёт счёт → Счёт отправлен → Оплачено →
Закрыто · успех
                                                    \
                                                     → Закрыто · потеря (на любом шаге)
```

Параллельно: **На паузе** — мета-статус с возвратом по дате.
