# Где что — навигация

Шапка вверху одинакова на всех экранах. Слева висит узкий рейл (56 px) с иконками разделов. Дальше — главный экран с тремя обычными состояниями: список (пул, инбокс), карточка (одна заявка) и сводный экран (дашборд).

![Дашборд: шапка, левый рейл, тело экрана](/images/docs/dashboard.png)

*Так выглядит главный экран РОПа. У менеджера набор кнопок в шапке короче и нет workspace-pill с общими ящиками, но устройство то же.*

## Шапка

| Элемент | Что делает |
|---|---|
| Логотип **MyLift CRM** | Возвращает на дашборд. |
| Workspace-пилл (`mail@…`) | Показывает основной общий ящик и количество активных. Зелёная точка — всё ок, янтарная — ошибка синка. |
| Поле поиска `⌕` | Поиск по своим заявкам — по теме, клиенту, артикулу. Жмёт `Enter` → переходит в пул с фильтром. |
| `Импорт из 1С` / `+ Заявка вручную` | Заглушки (Phase 2), пока не работают. |
| `🔔` колокольчик | Личные уведомления: вам назначили заявку, ваша заявка просрочена и т.п. Цифра — число непрочитанных. |
| Иконка-искра справа | **Связь с создателем.** Открывает форму для сообщения о проблеме или предложении. |
| Аватар | Профиль, мои обращения, выход. |

## Левый рейл

Иконки сверху вниз: **Дашборд**, **Заявки**, **Каталог**, **Счета**. Активный раздел подсвечен красной полоской слева.

Внутри раздела заявки рейл расширяется в полноценный list+detail: слева пул, справа карточка выбранной заявки.

## Основные разделы

- **Дашборд** — ваш сводный экран: сколько заявок в работе, сколько просрочено, KPI по неделе.
- **Заявки** — ваш пул и карточки заявок. Здесь вы проводите большую часть времени.
- **Каталог** — справочник позиций из 1С (read-only). Можно искать товары без привязки к заявке.
- **Счета** — список выставленных счетов. Менеджер видит только свои.

Документация открывается через `?` в шапке. Связь с создателем — через иконку-искру рядом.
