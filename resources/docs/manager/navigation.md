# Где что — навигация

В системе есть три постоянных элемента: верхняя шапка (48 px), узкий левый рейл (56 px) и главная область — в зависимости от раздела это таблица, карточка или сводный экран.

## Верхняя шапка

Слева направо:

| Элемент | Что делает |
|---|---|
| Логотип **MyLift CRM** | Возвращает на Дашборд. |
| **Workspace-пилл** (`mail@…`, зелёная/янтарная точка) | Только для РОПа / директора / секретаря / админа. Показывает основной общий ящик и количество активных. Менеджеру не показывается. |
| **Дашборд / Заявки** | Текущий раздел подчёркнут красной полоской. У РОПа добавляются ещё пункты: Правила почты, Авто-отклонённые, Менеджеры, Настройки. |
| Поле **поиска** (`⌘K`) | Поиск по своим заявкам — по теме, клиенту, артикулу. Жмёт `Enter` → переходит в пул с фильтром `?q=…`. |
| **Импорт из 1С** / **+ Заявка вручную** | Заглушки Phase 2, пока не работают. |
| **🔔 колокольчик** | Личные уведомления (вам назначили заявку, ваша просрочена). Число — непрочитанные. |
| **❓ круг с вопросом** | Эта документация. |
| **▲ треугольник с глазом** (красный, справа) | **Связь с создателем.** Открывает форму обращения. |
| **Аватар** | Профиль, Документация, Мои обращения, Выход. У админа — ещё «Обращения · инбокс». |

## Левый рейл (56 px)

Иконки сверху вниз:

| Иконка | Раздел | Кто видит |
|---|---|---|
| ⌂ | **Дашборд** | Все |
| ≡ | **Заявки** | Все |
| ⌕ | **Поиск по каталогу** | Все (read-only справочник из 1С) |
| ✉ | **Почта** | РОП / секретарь / директор / админ — менеджеру не показывается |
| ₽ | **Счета** | Все (менеджер — только свои) |

Под активными — серые заглушки Phase 2: **Правила**, **KB**, **Поставщики**, **SLA**. Внизу — **⚙ Настройки** (тоже Phase 2). Активный раздел подсвечен красной полоской слева.

## Основные разделы

- **Дашборд** — сводка: сколько у вас в работе, сколько просрочено, KPI за период (7/30/90 дней). У РОПа версия полнее: загрузка команды, воронка, поток входящих по часам, лента распределения.
- **Заявки** — пул и карточки заявок. Здесь вы проводите большую часть времени. См. [Мой пул заявок](/docs/manager/pool) и [Работа с заявкой](/docs/manager/request-lifecycle).
- **Поиск по каталогу** — найти позицию в каталоге 1С без привязки к заявке. Тот же hybrid-поиск (артикул + полнотекст + векторный), что и в карточке заявки.
- **Счета** — список всех выставленных счетов. Менеджер видит только свои.

> Дизайн-кит, по которому система собиралась, лежит в `design/ui_kits/crm/`. Если что-то в этих гайдах смотрится «теоретически» — сообщите через ▲ в шапке, я перепишу под фактический UI.
